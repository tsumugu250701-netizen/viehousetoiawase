Viehouse 画像差し替え用ファイル

このZIPを解凍して、GitHubのリポジトリ直下にアップロードしてください。

入っているもの：
- public/images/provence-hero.png
- public/images/smart-hero.png
- lib/content.ts

手順：
1. ZIPを解凍
2. GitHubの「Add file」→「Upload files」
3. 解凍した public フォルダと lib フォルダをそのままドラッグ
4. Commit changes
5. Vercelが自動で再デプロイ

注意：
GitHub上に既に lib/content.ts があるので、上書きになります。
